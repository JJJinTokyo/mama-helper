😢 宝宝哭了

以下是最常见的8种原因和对应信号，对照查看，找到最像的情况

使用方法：从第一张「紧急排查」开始看。如果不符合，继续往下找——对照每张卡片的「信号特征」，找到最像宝宝当前表现的那张。
🚨
先看这里：需要立即就医
有以下任意一种情况，立刻去急诊
  • 哭一阵停一阵，间歇期宝宝萎靡、脸色发白，或大便变果酱色 → 高度怀疑肠套叠，立即急诊
  • 哭声持续尖叫 + 发热 → 立即急诊
  • 3月龄以下任何发热（≥38°C） → 立即就医，不要等
  • 吐出黄绿色液体 → 立即就医
以上情况排除后，再往下继续对照。
😋
饿了
信号特征
  • 不停吃手，或嘴唇做吸吮动作
  • 用手指轻碰脸颊，宝宝立刻转头朝向手指
  • 头左右扭动，像在找什么
怎么做
  • 直接喂奶，不用等
  • 喂后立刻停哭、表情放松 = 确认是饿了
😪
困了
信号特征
  • 揉眼睛，或用手不停蹭脸
  • 打哈欠
  • 眼神开始涣散，不再盯着你
  • 逗她，她把头转开不理你
怎么做
  • 停止逗玩，带到安静暗处
  • 开白噪音，轻拍背
  • 越哄越哭 = 过度疲劳，停止所有刺激
🍑
尿布湿了
怎么确认
  • 摸尿布——鼓胀、沉甸，明显比之前重
  • 或者已经过了2小时以上没换
怎么做
  • 换上干净尿布
  • 顺便检查衣服里有没有线头或标签顶到皮肤
  • 检查是否有尿布疹，有红疹可用护臀霜
🌡️
太热或太冷
怎么判断（摸颈背，不是手脚）
  • 颈背出汗、发烫 → 太热了，减少一层衣物
  • 颈背偏凉 → 太冷了，加一层薄衣
  • 手脚凉是正常末梢循环，不代表冷
参考室温
  • 建议室温：冬 20–24°C · 夏 25–28°C
  • 新生儿比大人多穿一层
🤗
需要抱抱（情感需求）
怎么判断
  • 把宝宝抱起来，贴近你的左胸（让她听到心跳）
  • 哭声在30秒内明显减弱或停止
  • 其他原因都排除了
怎么做
  • 贴近心跳 + 轻拍背 + 低声哼唱
  • 不用担心"宠坏"——3月龄内及时回应是建立安全感的关键
💨
想打嗝（喂奶后）
信号特征
  • 刚喂完奶不久（30分钟内）
  • 发出短促的"嗯嗯"声
  • 身体往前弓，表情痛苦
怎么做
  • 竖抱，让宝宝趴在你肩上
  • 空心掌轻拍背部
  • 5–10分钟没拍出来可先放下，稍后再试
🫧
胀气 / 肠绞痛
最常见于傍晚，生后2周到4个月是高发期
信号特征（有一种就可能是）
  • 双腿不停蜷起来，又踢出去
  • 肚子摸起来硬鼓鼓的
  • 放屁后哭声明显减弱
  • 傍晚5–10点特别爱哭，其他时间正常
缓解方法
  • 顺时针轻按摩肚子（画圆圈）
  • 飞机抱：宝宝趴在你前臂，肚子朝下轻摇
  • 热毛巾敷肚子（不烫手为准）
  • 下次喂奶后充分拍嗝，减少吞气
好消息：4个月后大多自愈。这是「PURPLE Crying / 黄昏泣き」——生后2周到5个月的正常发育现象，不是你的问题。
⚠️ 警惕：如果哭的间歇期宝宝萎靡、脸色发白，或大便变果酱色，立即去急诊（肠套叠）。
🤷
都不像？暂时找不到原因
  • 抱着安抚，开白噪音，观察体温和精神状态
  • 如果持续哭闹超过3小时，建议就医
你已经尽力了，这不是你的问题。有时候宝宝就是需要哭一会儿，陪着就够了。
